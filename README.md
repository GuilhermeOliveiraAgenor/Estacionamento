# Estacionamento
Um sistema de estacionamento com controle dos veículos, clientes, sessões. Ainda por cima, o sistema tem envio de email com informações das sessões para o cliente, controle dos funcionários, relatório sobre as sessões e relatório em pdf. Clique para conferir.

Ao baixar o código, é necessário mudar a conexão do banco de dados no qual desejar trocar.
